JavaScript 物件(3) - 比較 陣列、JSON

發表於 2020-08-04 更新於 2020-10-23 分類於 JavaScript Disqus：

Array 陣列

  1. 也是物件型別，但是沒有傳參考的特點。

  2. 定義：用「[]」中括號，「不用屬性，直接填值」而且裡面可以放各種值。

  3. 取值：中括號 + (索引)位置。(注意從 0 開始計算)。

  4. 新增值：有 3 種例子。

  5. 陣列長度：array.length 陣列裡面一共有幾個值，不是從 0 開始算。

  6. 常搭配 for 取出所有值

JSON

  • 一種獨立語言的文字格式

  • 格式：屬性、值都用「雙引號」是字串形式。

  • 跟物件很像但是不同：物件可以用單引號。

  • 常跟 Ajax 搭配抓資料
    使用 Ajax 抓到 JSON 字串資料，在轉成物件來方便應用，很多工具會自動轉成物件的方式。(在網頁開發者工具的 Network 的 Response 可以看到資料是 JSON 格式)
