Hexo 常用指令

發表於 2020-06-25 更新於 2020-06-29 分類於 Hexo Disqus：

生成靜態網頁 (全寫hexo generater)

會產生public檔案

本地端預覽 (全寫hexo serve)

deploy 文件生成後立即部屬上去遠端網站

(可以省略 hexo g)

watch 監視文件變動 (-w)

網站存檔時自動升成新的靜態網頁
(適合測試網站時使用)

clean 清除

清除暫存(db.json)和已生成的靜態文件(public)

更換主題時生成有問題可以使用

錯誤訊息紀錄

會生成一個 debug.log 檔案裡面

新增文章-草稿文章

雖然一開始比較想要用手動，但是指令比較快還是需要熟悉，而且會自動把每頁需要的…建立好。

(手動)
在 source 的資料夾建立 _drafts 資料夾，新增一份.md。

(指令)
新增草稿文章

新增頁面
